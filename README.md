# 八束穂バス時刻表アプリ

金沢工業大学扇が丘キャンパスから八束穂キャンパスへ向かう際のバス時刻表を表示するGUIアプリケーションです。

## 概要

このアプリケーションは、八束穂キャンパス内の23号館と65号館間を運行するバスの時刻表を表示し、次のバスの出発時刻と到着予定時刻、および現在時刻からの残り時間をリアルタイムで表示します。

## 機能

- **リアルタイム時刻表示**: 現在の日付と時刻を表示
- **ルート選択**: 23号館→65号館、65号館→23号館のルートを選択可能
- **次のバス情報表示**: 
  - 次のバスの出発時刻
  - 到着予定時刻
  - 現在時刻からの残り時間（カウントダウン）
- **休暇・祝日対応**: 夏季休暇、春季休暇、祝日などの運休日を自動判定
- **曜日別時刻表**: 平日、土曜日、休暇中の時刻表を自動切り替え
- **日本語フォント対応**: 各OSに適した日本語フォントを自動選択

## 使用方法

### 実行方法

```bash
python bus_time.py
```

### 操作手順

1. アプリケーションを起動
2. ルートを選択（23号館→65号館 または 65号館→23号館）
3. 次のバスの情報が自動的に表示されます

## システム要件

- Python 3.x
- tkinter（Python標準ライブラリ）
- 対応OS: Windows, macOS, Linux
※動作確認はLinuxのみで行っています。

## 時刻表データ形式

CSVファイルの形式：
```
出発時,出発分,到着時,到着分
8,20,8,50
8,40,9,10
...
```

- **祝日**: 4月29日、5月3日〜6日、6月1日、7月19日〜21日、8月2日、8月7日〜17日、8月23日、8月30日、9月13日〜15日、9月21日〜23日、10月13日、10月18日〜19日、11月3日、11月24日、12月27日〜31日、1月1日〜6日、1月12日、2月11日、2月23日、3月20日

## 注意事項

- 時刻表データは金沢工業大学の公式情報に基づいています
- 実際の運行状況は大学の公式発表をご確認ください
- 祝日や特別な行事により運行が変更される場合があります
